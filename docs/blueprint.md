# **App Name**: PendencyFlow

## Core Features:

- Static Page Navigation: Implement client-side navigation for a multi-page static site structure, including Home, Features, About, Demo/Pilot Results, Pricing, Contact, and Blog, ensuring an SEO-friendly setup with Next.js.
- Responsive UI & Content Presentation: Deliver a mobile-first and fully responsive user interface to showcase the app's tagline, features (with cards and icons), mission statement, demo results, pricing tiers, and contact information across all device sizes.
- Informational Calls-to-Action: Display static, non-functional Call-to-Action buttons for 'Request Demo', 'Download App', 'App Store', and 'Play Store' to guide users, reflecting a frontend-only MVP.
- Static Forms & Contact Info: Present a static contact form and display essential contact details (email, support text) and social media icons in the footer without any backend submission logic.
- Dummy Data & Testimonials: Integrate placeholder data such as static charts, simulated 'delay reduction' text, and dummy testimonials to illustrate the app's impact and success stories in the Demo/Pilot Results section.
- AI Concept Visualizer Tool: Utilize static, pre-rendered examples of AI-generated insights or outputs to visually demonstrate how features like case duration prediction and urgent case prioritization work within the PendencyFlow application.
- SEO & Performance Optimization: Structure HTML semantically and optimize all static assets (images, CSS, JavaScript) to ensure high performance, quick loading times, and excellent search engine visibility when deployed on Firebase Hosting.

## Style Guidelines:

- Primary color: A deep, authoritative navy blue (Hex: #0B378E) for headings, primary text, and key branding elements, evoking trust and professionalism.
- Background color: A serene, very light blue-grey (Hex: #F0F2F4) serving as the main background, providing a clean canvas that subtly hints at the primary blue hue.
- Accent color: A clear, vibrant sky blue (Hex: #52C9E0) for Call-to-Action buttons, interactive elements, and highlights, offering excellent contrast and a sense of innovation.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective, and highly legible aesthetic suitable for legal documentation and tech interfaces.
- Use a consistent set of clean, minimalist outline icons for features and functional elements, enhancing readability and visual clarity in a legal-tech context.
- Implement a mobile-first, responsive grid system with generous spacing and clear sectional divisions to ensure optimal content hierarchy and usability on all screen sizes.
- Apply smooth yet subtle animations for page transitions, hover states on interactive elements, and progressive content reveals to enhance user experience without causing distraction.
